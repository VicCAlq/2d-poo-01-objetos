# Tags HTML

O React usa html clássicas (button, image, div, p, h1, entre outras) para compôr seus componentes. Vamos relembrar algumas das principais tags:

| Tag(s) HTML correspondente(s) | Descrição |
|-------------------------------|-----------|
| `<div>`, `<section>`, `<header>`, `<footer>`, `<aside>` | Elementos organizadores, para divisão de conteúdo e formatação de como são exibidos (em linha, coluna, etc) |
| `<p>`, `<h1>`, `<h2>`, `<h3>`, etc | Equivalente a tag de parágrafo. React Native não tem tags separadas para tamanhos de textos diferentes, estes devem ser criados com a estilização dos mesmos |
| `<button>` | Cria um botão com possibilidade de interação |
| `<img>` | Elemento para inserir uma imagem | 
| `<input type="text">` | Campo de inserção de texto |
| `<ol>`, `<ul>` | Elemento para criação de listas simples |
| `<input type="checkbox">` | Chave de liga-desliga |
| `<dialog>` | Cria uma janela flutuante sobre a página atual |
| `window.alert()` | Cria um alerta em uma janela flutuante simples |
| `.css` | Equivalente a um arquivo de estilos CSS |
